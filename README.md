# ViaSur 🚚

## [Enlace al sitio web de la propuesta](https://figueroawa.github.io/viasur/index.html)

## ¿Qué es?

ViaSur es nuestra propuesta al desafío que se nos planteó en la Hackathon del Innovation and Technology
Summit Puerto Varas 2026.

En este repositorio está el prototipo de la aplicación, puedes encontrar los detalles de la propuesta en el sitio web.

## ¿Cómo probar el prototipo?

Puedes clonar este repositorio o descargarlo como archivo zip y abrir el archivo ***home.html*** con tu navegador web.
